En esta actividad aprendí a recibir datos enviados por los usuarios mediante peticiones POST utilizando Express. Comprendí la importancia del formato JSON para intercambiar información entre aplicaciones. También entendí cómo funciona req.body para acceder a los datos enviados por el cliente. Este tipo de aplicaciones permite registrar reportes de manera organizada y eficiente. Las plataformas tecnológicas pueden ayudar a mejorar la comunicación entre los ciudadanos y las instituciones. Además, facilitan el seguimiento de incidencias y solicitudes. Esta práctica fortaleció mis conocimientos sobre el desarrollo de APIs con Node.js y Express. Considero que estos conocimientos serán útiles para futuros proyectos.
